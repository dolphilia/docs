使い方 ​

開発環境のセットアップ ​

前提条件 ​

  • Node.js 16.0.0以上
  • pnpm 8.0.0以上

インストール ​

リポジトリをクローンした後、以下のコマンドを実行して依存関係をインストールします。

bash
pnpm install
1

開発サーバーの起動 ​

メインサイトの開発サーバーを起動するには、以下のコマンドを実行します。

bash
pnpm dev
1

特定のプロジェクトの開発サーバーを起動するには、以下のコマンドを実行します。

bash
pnpm --filter project1 dev
1

ビルド ​

すべてのプロジェクトをビルドするには、以下のコマンドを実行します。

bash
pnpm build
1

特定のプロジェクトをビルドするには、以下のコマンドを実行します。

bash
pnpm --filter project1 build
1

多言語対応 ​

言語の切り替え ​

言語を切り替えるには、URLのパスを変更します。

  • 日本語: /
  • 英語: /en/

翻訳の追加 ​

新しい言語を追加するには、以下の手順に従います。

  1. .vitepress/config.jsのlocalesに新しい言語を追加します。
  2. 各プロジェクトに新しい言語のディレクトリを作成します。

バージョン管理 ​

バージョンの切り替え ​

バージョンを切り替えるには、<VersionSelector />コンポーネントを使用します。

新しいバージョンの追加 ​

新しいバージョンを追加するには、以下の手順に従います。

  1. 各言語ディレクトリに新しいバージョンのディレクトリを作成します。
  2. .vitepress/config.jsのversionsに新しいバージョンを追加します。
