## Overview

This project is a sample project for the VitePress monorepo documentation site. It was created to demonstrate how to manage multiple VitePress projects in a monorepo structure.

## Features

- **Monorepo Structure**: Efficient project management using pnpm workspaces
- **Shared Components**: Reusable components across all projects
- **Multilingual Support**: Easy management of Japanese and English documentation
- **Version Management**: Easy switching between document versions
